Appointful

Appointful é uma API feita em Node.js para gerenciar consultas odontológicas. Nela você pode listar dentistas e consultas, agendar consultas, cancelar consultas e muito mais.

  • Apliquei padrões de projeto como Factory, Adapter, Composite e muitos outros;
  • Apliquei princípios SOLID;
  • Escrevi testes unitários e end-to-end com Vitest para garantir que tudo funciona perfeitamente;
  • Documentei as rotas da API com Swagger;
  • Utilizei Prisma com PostgreSQL para persistir e recuperar dados;
  • Fiz deploy do banco de dados e da aplicação em produção com Render.com;

React Native Sortable Gallery

Um componente React Native que permite ordenar uma galeria de imagens. Este componente permite que você acompanhe a ordem das suas imagens e seja capaz de organizá-las de uma maneira fácil e performática.

  • Desenvolvi todo o projeto;
  • Utilizei React Native com Expo para construir a aplicação;
  • Utilizei React Native Reanimated e React Native Gesture Handler para permitir que usuários arrastem e soltem imagens de uma maneira limpa e performática;
  • Utilizei ESLint e Prettier para garantir qualidade de código e impor padrões de codificação;
  • Utilizei Husky e lint-staged para garantir que os padrões de código são mantidos antes de enviar para produção;
  • Utilizei TypeScript para garantir segurança de tipos;
